# Auliviet's dotfiles
A simple configuration for git, tmux, vim, and zsh.

## Git
Source version control.


## Tmux
Terminal multiplexer.

Uses:
- cattpuccin (theme)
- tmux-sensible (defaults)
- tpm (plugin manager)


## VIM
Text editor.

Uses:
- catpuccin (theme)
- vim-sensible (defaults)


## ZSH
Shell.

Uses:
- fzf
- Oh-my-zsh
- zsh-autosuggestions
- zsh-completions


## Getting started
Installation of the dotfiles is managed by ansible.
